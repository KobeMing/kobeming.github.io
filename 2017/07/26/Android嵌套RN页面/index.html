2. 创建容器Activity

创建容器Activity有两种方案。一:继承ReactNative提供ReactActivity。二:继承Android原生Activity,然后自己实现ReactNative提供的DefaultHardwareBackBtnHandler, PermissionAwareActivity接口.

建议使用第二种方式,这样可以实现预加载需求.

3. 创建ReactRootView
